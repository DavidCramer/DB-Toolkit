Build additional Content Management Structures and Database Applications right into your WordPress Site.

== Description ==

DB-Toolkit allows you to create user interfaces into a database table, both existing and custom.

V1 is still very much in development. It cannot be used in production at this stage. Please feel free to help out with its development.

== Installation ==

1. Upload the plugin folder 'db-toolkit' to your `/wp-content/plugins/` folder
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Select DB Toolkit from the menu group
